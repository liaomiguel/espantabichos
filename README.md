# Espantabichos — Control Sónico de Plagas

![Espantabichos Banner](https://img.shields.io/badge/Status-Premium-blueviolet?style=for-the-badge)
![Web Audio API](https://img.shields.io/badge/Powered%20by-Web%20Audio%20API-orange?style=for-the-badge)

**Espantabichos** es una solución acústica avanzada de grado premium diseñada para la disuasión de plagas mediante el uso de frecuencias subsónicas y ultrasónicas. Aplicando principios de ingeniería acústica, la aplicación utiliza algoritmos de frecuencia variable para evitar que las plagas se habitúen al sonido.

## 🚀 Características Principales

- **Algoritmos Anti-Habituación**: Modulación de frecuencia en tiempo real (Random & Sine sweep) que evita que el sistema pierda efectividad con el tiempo.
- **Modos de Protección Específicos**:
  - 🐁 **Roedores**: Optimizado (28 - 42 kHz) para ratas y ratones.
  - 🐜 **Insectos**: Frecuencias agresivas para hormigas y cucarachas.
  - 🪰 **Voladores**: Interferencias constantes para moscas y mosquitos.
  - 🌀 **Barrido Total**: Cubre todo el espectro disuasorio de forma aleatoria.
- **Monitor Maestro**: Visualización en tiempo real de la frecuencia emitida y el tiempo de actividad.
- **Visualizador de Ondas**: Feedback visual dinámico de la señal emitida.
- **Diseño Premium**: Interfaz moderna inspirada en ElevenLabs, optimizada para modo oscuro y visualización clara.

## 🛠️ Tecnologías

- **Core**: HTML5 Semántico.
- **Estilos**: CSS3 moderno con variables (Custom Properties) y diseño flexible (Grid/Flexbox).
- **Lógica**: JavaScript Vanilla (ES6+).
- **Motor de Audio**: Web Audio API para generación de señales pura sin necesidad de archivos de audio externos.

## 💻 Instalación y Uso

No requiere instalación de dependencias ni servidores complejos:

1. Clona el repositorio o descarga los archivos.
2. Abre `index.html` en cualquier navegador moderno (Chrome, Edge, Safari preferentemente por soporte de Web Audio API).
3. Selecciona un **Objetivo de Protección**.
4. Haz clic en **INICIAR EMISIÓN**.

> [!IMPORTANT]
> Para máxima efectividad, evite colocar el dispositivo emisor detrás de muebles o cortinas que puedan absorber la energía acústica.
